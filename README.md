# React Native Demo

# Install

```
$ npm install
```


# Running on Android:

```
$ react-native run-android
$ adb reverse tcp:8081 tcp:8081   # required to ensure the Android app can
$ adb reverse tcp:8080 tcp:8080   # access the Packager and GraphQL server
```
# Running on iOS:

```
$ react-native run-ios
```
